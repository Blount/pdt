Setting Workbench Preferences

  1. If you still see the Eclipse Welcome page, click the arrow icon to begin using Eclipse. Select the menu item Window > Preferences to open the workbench preferences.

  2. Go to the tree pane and select the "root" PHP category. A drop down menu will appear.

  3. Open the drop down menu and select the PHP Version to use from the drop down menu.

  4. In order to work with ASPs, check the Use ASP checkbox.

  5. Select and configure the remaining Preferences and/or the Project Specific Preferences.
