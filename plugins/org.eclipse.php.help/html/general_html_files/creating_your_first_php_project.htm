Creating Your First PHP Project

A wizard is used to help create your PHP projects.

  1. Bring up the New Project Wizard by clicking File | New | Project or by placing the cursor into the Navigator pane, right clicking and selecting New | Project. Select PHP. The New Project Wizard will appear.

Figure: 1 -  New Project

  1. Click Next. The PHP Project dialog will appear.
