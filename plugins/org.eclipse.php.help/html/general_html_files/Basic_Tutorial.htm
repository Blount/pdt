Basic Tutorial

The Zend PHP Eclipse IDE is a complete development environment for editing, debugging and optimizing code for PHP applications. The fully integrated user interface provides features that help streamline the development process.

This Zend PHP Eclipse IDE user Guide is a task-oriented guide to procedures typically performed by PHP developers. Throughout this online help you will find descriptions of the IDE's features and functionality and how they can be used to streamline the development process.

This User Guide is intended for the Zend PHP Eclipse IDE 1.0. Use the table of contents or the index to locate topics of interest.
