The PHP Integrated Development Tool

The PHP Integrated Development Tool (PDT) refers to the desktop development environment. This environment provides a number of tools that are designed to work together in a seamless fashion. These include creation, management, and workspace navigation resources.

Each PDT window contains one or more perspectives.  These contain specific, user-definable views, editors and tools for managing and working with certain menus and toolbars.  Multiple windows can exist on the desktop at any given time.
