The Integrated Development Environment (IDE/Workbench)

The Integrated Development Environment (IDE) refers to the desktop development environment. This environment provides a number of tools that are designed to work together in a seamless fashion. These include creation, management, and workspace navigation resources.

Each IDE window contains one or more perspectives.  These contain specific, user-definable views, editors and tools for managing and working with certain menus and toolbars.  Multiple windows can exist on the desktop at any given time.
