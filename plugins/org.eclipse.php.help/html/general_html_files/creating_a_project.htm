Creating a Project

To create a project:

  1.  On the main menu bar, click File > New > Project. The New Project wizard will open.

 

Figure: 1 -  New Project

  1. Select the category PHP and then PHP Project. Click Next. The Project Creation dialog will open.

 

Figure: 2 -  Project Creation

  1. Enter a name for your new project into the Project Name field. The project that you create will map to a directory structure in the file system. The default file system location is displayed (dimmed) in the Directory field. If you want to create the project and its contained resources in a different location, clear the Use default location checkbox and specify the new location.

  2. If you want the new project to be dependent on one or more other projects, click Next. The PHP include path dialog will open. Select the projects to be referenced.

 

Figure: 3 -  New Project

  1. Click Finish. The new project is listed in one of the navigation views.

The General > Perspectives > preference page allows you to specify the perspective behavior when a new project is created
