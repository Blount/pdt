Introduction

The PHP Eclipse IDE user Guide is a task-oriented guide to procedures typically performed by developers. Throughout this online help you will find descriptions of the PHP Eclipse IDE Features and functionality and how they can be used to streamline the development process using PHP Eclipse IDE.

 

The PHP Eclipse IDE is a complete development environment for editing, debugging and optimizing code for PHP applications. The fully integrated user interface provides features that help streamline the development process.

 

This User Guide is intended for PHP Eclipse IDE 1.0. Use the table of contents or the index to easily locate topics of interest.
