Project Specific Settings

  1. In order to set preferences (Editor, Debug, Formatter, etc.), for a specific project open Windows > Preferences > PHP > "preference".

  2. Click the link: Configure Project Specific Settings.... at the top right of the dialog. The Project Specific Configuration dialog will appear.

 

  1. Select the project and click OK; the Properties for <project> dialog will appear.

  2. Click the select icon (). The list of available properties will appear.  

  1. Click and select the type of property to set. The corresponding dialog will appear.

  1. Set the preferences as required and click OK.
