Filtering Elements

To filter elements:

  1. On the Package Explorer toolbar, click the Menu button and choose Filters. The PHP Element Filters dialog will appear.

              Figure: 1 -  Filters

  1. Select or clear the filters that you want to apply to the view (read the Filter description for a description of the filter).

  2. Optionally, you can select patterns for filtering the view:

    • Select the Name filter patterns checkbox at the top of the dialog.

    • In the text field below, specify the desired patterns (names matching any of the patterns will be hidden).
