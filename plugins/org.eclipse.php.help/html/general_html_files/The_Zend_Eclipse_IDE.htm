The Eclipse PHP IDE

The Eclipse PHP IDE is a functionality rich, PHP Integrated Development Environment.

It is an Eclipse.org project designed to meet PHP’s developing needs and to support newer versions of PHP (currently PHP 5) as well as previous versions. The Eclipse PHP IDE contains advanced editing capabilities, debugging and code inspection.

An understanding of the many features and functionality is required in order to fully benefit from the Eclipse PHP IDE’s capabilities and features. This User Guide has been designed to describe Eclipse PHP IDE from the basic user interface layout to more complex features, each in their own dedicated chapter.

The functionality of each screen is elaborated in the online Help, to provide context sensitive instructions on how to work with the Eclipse PHP IDE.

Audience

Eclipse PHP IDE is an integrated development environment for developers of PHP applications.
