The PHP Integrated Development Environment

The PHP IDE is a functionality rich, PHP integrated development environment developed by Zend and IBM.   It is an Eclipse.org project designed to meet PHP’s developing needs and to support the current version of PHP (currently version 5.x) as well as previous ones.

The PHP IDE contains advanced editing capabilities, debugging and code inspection.

This User Guide describes the PHP IDE from the basic user interface layout to more complex features. The functionality of each screen is elaborated in the online Help in order to provide context sensitive instructions..

Audience

The PHP IDE is an integrated development environment for developers of PHP applications.
