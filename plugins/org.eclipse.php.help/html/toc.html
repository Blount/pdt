The PHP Developer IDE

The PHP IDE is a functionality rich, PHP Integrated Development Environment developed by Zend and IBM. It is an Eclipse.org project designed to meet PHP’s developing needs and to support newer versions of PHP (currently PHP 5) as well as previous versions.

The PHP IDE is based on the well known Zend Studio and contains advanced editing capabilities, debugging and code inspection. This User Guide describes the PHP IDE from the basic user interface layout to more complex features, each in their own dedicated chapter. The functionality of each screen is elaborated in the online Help, and provides context sensitive instructions on how to work with the PHP IDE.

Audience

The PHP IDE is an integrated development environment for developers of PHP applications.
