The PDT (PHP Development Tool)

The PDT is an Eclipse.org project designed to meet PHP’s developing needs and supports the current version of PHP (currently 5.x) as well as previous ones.

The Eclipse PDT is a functionality rich, PHP tool containing advanced editing capabilities and code inspection.

This User Guide describes the PDT from the basic user interface layout to more complex features. The functionality of each screen is elaborated in the online Help in order to provide context sensitive instructions..

Audience

The PDT is a development tool for developers of PHP applications.
