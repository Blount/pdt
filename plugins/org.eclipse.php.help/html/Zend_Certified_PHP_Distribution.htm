Zend PHP Web Server

 

You can benefit from Zend's expertise by using the Zend PHP Web Server, which comes bundled with the Zend Debugger, in order to perform optimal debugging and profiling.

 

It includes the most reliable and up-to-date version of PHP, tested PHP extensions, database drivers and other enhancements. The Zend PHP Web Server comes bundled with Zend Framework (the leading open-source PHP framework), Apache and MySQL. The Zend PHP Web Server also supports IIS web servers. The complete stack is built, tested and benchmarked to ensure optimum reliability and performance for Linux, Windows, Unix, i5/OS and Macintosh servers.

 

See the Zend PHP Web Server site to learn more about and download the Zend PHP Web Server.
