Creating PHP Working Sets

PHP Working Sets can be created from a variety of locations where working sets can be selected (see PHP Working Sets for more information.)

 

This procedure describes how to create PHP working sets from the Window Menu in the PHP Perspective.

 

 

To create a PHP Working Set:

  1. Go to Window | Working Sets | Edit..
    The Select Working Sets dialog will open.

  1. Click New.
    The New Working Set dialog will open.

  2. Select PHP and click Next.
    The New PHP Working Set dialog will appear.

  3. Enter a name for the Working Set.

  4. Mark the checkbox next to the projects/files/folders to be included in the working set.

  1. Click Finish.

Your new PHP working set will be added to the Select Working Sets dialog and will be available for selection for Working Sets actions.
