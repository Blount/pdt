Creating Zend Framework Elements

 

Once you have created a Zend Framework Project, you can add Model, View, View Helper and Controller files to your project using the New Zend Model, Zend View, Zend View Helper and Zend Controller wizards. These files should be placed within a 'Zend Module', which enables you to organize your MVC files.

Note:

When working with Zend Framework files, you should open the Zend Framework perspective by going to Window | Open Perspective | Zend Framework.

To learn how to create a Zend Module, see Creating a Zend Module File.

To l earn how to create a Zend Model, see Creating a Zend Model File.

To learn how to create a Zend Controller, see Creating a Zend Controller File.

To learn how to create a Zend View, see Creating a Zend View File.

To learn how to create a Zend View Helper, see Creating a Zend View Helper File.
