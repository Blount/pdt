PHP Project Outline View

 

The PHP Project Outline view displays a list of Soap Clients, Constants, Classes and Functions for all files within the selected project.

Selecting an element in the PHP Project Outline view will open the relevant file in the editor.

PHP Project Outline view Toolbar commands

Icon

Name

Description

Collapse All

Collapses the list of elements

Link with Editor

If selected, elements will immediately be displayed in the editor when they are single-clicked. If unselected, elements will be displayed in the editor when they are double-clicked.

Note:

The PHP Project Outline View is displayed by default as part of the PHP Perspective. To manually open the view, go to Window | Show View | Other | PHP Tools | PHP Project Outline.
