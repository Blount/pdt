Creating a Zend Model File

This procedure describes how to create a new Zend Model file.

A Zend Model is a class for working with the "model" portion of the model-view-controller pattern. The Zend Model manages the information and data manipulation in your application.

 

 

To create a new Zend Model file:

  1. In PHP Explorer view, right-click the models folder in your Zend Framework Project and select New | Zend Framework Item | Zend Model.
    The New Zend Model Wizard will be displayed.

  2. Ensure the source folder is correct and enter the required file name.

  3. Click Next.

  4. In the Select PHP Template dialog, ensure the correct template is selected for the item you are creating.

  5. Click Finish.

A new Zend Model file will be created with the relevant template.

The new file will be displayed in the MVC Oultine view.
