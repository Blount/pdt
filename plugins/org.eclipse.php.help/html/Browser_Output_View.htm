Browser Output View

 

The Browser Output view will show the output of the script to a browser. This will be updated as the debugging process continues.

Note:

The Browser Output View is displayed by default as part of the Debug Perspective. To manually open the view, go to Window | Show View | Other | PHP Tools | Browser Output.
