Purpose and Usage

 

Debugging Local PHP Files (PHP Scripts)

This tutorial demonstrates how to easily deploy and profile a simple Web application with PDT and Zend Server.

Note:

You must have Zend Server installed on your local machine before commencing this tutorial.

 

 

  1. If you have not yet configured your Zend Server integration, open the Servers View and click the Auto Detect Zend Server button on the view's toolbar.
    A dialog...

  2. A popup ballon will appear in the bottom-right corner of the window indicating that a Zend Server installation has been detected and configured.

  1. Click the icon to close the balloon.

  2. From the menu bar, go to File | New | PHP Project.
    The New PHP Project dialog is displayed.

  3. Enter 'ZendServerProfiling' in the project name field.

  4. Under the contents category, ensure the 'create project on a local server' is selected.
    The directory should automatcially point to the document root of your Zend Server.

  5. Click Finish.
    The project will be created on your Zend Server's document root.

  6. Create a new PHP file in the project by right-clicking the project in PHP Explorer view and selecting New | PHP File

  7. Name the file c-to-f_convert.php and click Finish.

  8. Copy pastecopy-paste the example code into the file. (Click here to see the code.)

  9. Save the file.

  10. Run the file by going to Run | Run As | PHP Web Page.
