What's New in PDT 2.0

The following new features are available in PDT 2.2:

  • Phar Integration

  • Mylyn Integration

 

The following new features are available in PDT 2.1:

  • PHP 5.3 Support:

  • Ability to create a PHP project based on PHP 5.3.

  • Syntax Coloring and Validation for PHP 5.3 in the editor

  • Code Assist for Namespaces

  • Code Assist for packages

  • Namespace grouping in PHP Explorer

The following new features are available in PDT 2.0:

  • PHP Build Path Configuration

  • Override Indicators

  • Type Hierarchy

  • Mark Occurrences

  • Open Type/Method

  • Code Assist Enhancements - Camel Case matches, Magic Members support

View the Zend Product Online Documentation for the most up-to-date information.
