Searching the Zend Framework Site

 

PDT's workbench contains a search box which allows you to quickly and easily retrieve information from the Zend Framework site, without having to leave the application.

 

This procedure describes how to search for information in the Zend Framework site.

 

 

To search the Zend Framework site:

  1. Enter a term in the Zend Framework search box in the bottom of the Workspace.

  

  1. Click the 'Go' button.
    A new browser window will open in PDT, displaying a list of relevant results from the Zend Framework site.

Note:

The browser will open according to the settings defined in Window | Preferences | Web Browser.

  1. Click the required result to be taken to the page containing the relevant information.
