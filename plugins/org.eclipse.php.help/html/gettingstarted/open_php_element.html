Open PHP Element

Use Open PHP Element to go to and open a PHP element (Class, Function or Constant) in an open file or project.

To open the Open Element dialog

  1. Select Navigate | Open PHP Resource from the main menu. The Open PHP Element dialog will open.

  2. Enter the resource name (you may use wildcards).

  3. Filter by type (Classes, Functions, Constants).

Figure: 1 -  Open PHP Element

 

  1. Resources that match the entry will be listed in the Matching Resources list. The Open PHP Element dialog lists resource names alphabetically according to the characters or strings entered into the text entry field at the top of the dialog.

  2. Select the resource desired and click OK to navigate to the resource.
