Configuring Perspectives

In addition to configuring the layout of your perspective you can also control several other key aspects of a perspective. These include:

  • The options available on the File > New submenu.

  • The options available on the Window > Open Perspective submenu.

  • The options available on the Window > Show View submenu.

  • Action sets (buttons and menu options) that show up on the toolbar and menu bar.

Configuring a Perspective

  1. Switch to the perspective that you want to configure.

  2. Select Window > Customize Perspective. The Customize Perspective dialog will open.

  3. Select the appropriate submenu from the drop down list.

Figure: 1 -  Customize Perspective

  1. Select the checkboxes of the elements you want to see on the specific  menu in the selected perspective.

  2. Items you do not select will still be accessible by clicking the Other menu option.

  3. Click OK.

  4. Select the Commands tab. The command groups available to the perspective will be displayed.

  5. Select the desired groups and click OK.
