System Requirements

PHP Eclipse PDT

Supported platforms:

  • Microsoft Windows 2000 (all) / XP Professional / 2003 Server family.
  • (Windows 2000 users should install Microsoft's Service Pack 3).

  • Linux glibc 2.1 / 2.2 / 2.3 (e.g. RedHat 6.x / 7.x / 8.x / 9.x / Fedora Core, Debian 2.2, SuSE 6.4 / 7.2, Mandrake 8.1 / 9 / 10, and others).

  • Mac OSX 10.2/3 G3, G4 or G5 processors.

Note:    The PDT will probably work under other platforms which support the Java 2 environment, however, it is not officially supported under platforms other than those listed above.

  • 500MHz processor (800MHz processor recommended)

  • 256MB RAM (512MB recommended)

  • 100MB of hard disk space
