Getting Started

                  ......with the Eclipse PDT

The Eclipse PDT is a complete development environment for PHP applications. The fully integrated user interface provides features that help streamline the development process.

The Online Help is a task-oriented guide to the Eclipse PDT. Throughout this online help you will find descriptions of the IDE's features and functionality and how they can be used to streamline PHP development process.

Starting

  1. Launch the PHP Eclipse IDE. You will see a dialog asking you to select the location of the workspace. This is the directory where your work will be stored.

  2. Click OK to accept the default location (check the checkbox to prevent this question from being asked again).

  3. After the workspace location is chosen, Eclipse will open to the Welcome screen.

  4. Click the arrow at the top right to enter the IDE. The Eclipse PDT will open.

This will be your work environment. It contains several panes, each of which can contain one or more perspectives. These are specific views, each of which contains a defined functionality, e.g., editors, navigators (the PHP Explorer and the Navigator), filters, etc.
