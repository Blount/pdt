Opening the IDE

When the PHP Eclipse IDE is first launched you will see a dialog that allows you to select where the workspace should be located. This is the directory where your work will be stored.

  1. Click OK to accept the default location (check the checkbox to prevent this question from being asked again).

  2. After the workspace location is chosen, Eclipse opens to the Welcome screen. Click the arrow at the top right to enter the IDE. The Eclipse IDE will open.

This will be your work environment. It contains several panes, each of which can contain one or more perspectives. These are specific views, each of which contains a defined functionality, e.g., editors, navigators (the PHP Explorer and the Navigator), filters, etc.

Perspectives
