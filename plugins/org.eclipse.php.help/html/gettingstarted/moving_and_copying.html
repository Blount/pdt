Moving and Copying PHP Elements

PHP Explorer

  1. Go to the PHP Explorer or the Navigator. Right click an element or file, etc. A drop down menu will appear.

Figure 1  - Right Click

  1. Select Copy or Delete, as required.

  2. Place the cursor in the location you want to copy or move the element or file to.

  3. Right click. Select Paste to place the element or file in the new location. The element will be Pasted to its new location

Move

  1. Go to the Explorer or the Navigator. Right click a file, etc. A drop down menu will appear.

  2. Select Move. The Move dialog will appear.

Figure: 2 -  Move

  1. Select the location you want to move the file to and click OK. The file will be moved to its new location.
