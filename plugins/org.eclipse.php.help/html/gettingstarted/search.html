Search

The Eclipse PHP IDE's search functions enable you to find declarations of Classes, Functions and Constants.

To open the Search dialog
  1. Click Search | Search from the toolbar menu, click the Search icon () or press Ctrl+H. The Search dialog will open. Select the PHP Search tab.

Figure 1 - Search Dialog

  1. The Search dialog includes the PHP Search tab with the following fields:

    • String field (supports the ' ? ' and ' * '  wildcards).
      • ' * '  matches any set of characters, including the empty string

      • '?'   matches any character

      • " \ " is the escape for a literal; to search for an asterisk, question mark, or backslash character, type a backslash before it to indicate that you are not using these characters as wildcards (e.g., "\*", "\?", or "\\")

    • Case-sensitive search (check box).
    • Previous search expressions - in a drop-down list (all values associated a previous search expression are restored together with the saved expression.
    • Search For: Classes, Functions or Constants. Click one to select the type of resource search to do.
    • Scope - Includes four (4) options to define the search's scope: 
      • Workspace - Searches all resources in the workspace.
      • Selected Resources - Searches only the selected resources (disabled when no selected resources are selected).
      • Enclosing Projects - Searches all enclosing projects belonging to the selected resource.
      • Working Set - Searches a pre-defined working set (click the a Choose button to select the working set).
Search Hotkeys
    • Esc, Alt + F4   Closes the Search dialog
    • Alt + a              Focuses the Search String field
    • Alt + i               Toggles the Case Sensitive checkbox
    • Alt + c              Checks the Search For Class radio button
    • Alt + f               Checks the Search For Function radio button
    • Alt + o              Checks the Search For Constant radio button
    • Alt + w             Scope : Workspace radio button
    • Alt+d                Scope : Selected Resources radio button
    • Alt + j               Scope : Enclosing Project radio button
    • Alt + k              Scope : Working set radio button
    • Alt + h              Scope : Choose working sets button
    • Alt + z              Opens the Customize Search page
    • Alt + s              Performs Search
