Configuring the Code Formatter

  1. Open Windows > Preferences > PHP > Formatter. The formatter dialog will open.

  1. Select either:

    • Indent Using Tabs (uses the OS's default tab settings), -or-

    • Indent Using Spaces. Select the Indentation Size; this is the number of spaces to be indented whenever a tab space is implemented

  1. Click Apply / OK to apply the setting and exit.
