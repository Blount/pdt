Hover Help

The PHP IDE enables Hover help while editing and includes the following elements:

Location

The file in which the PHP element was declared

Short Description

This is the short description taken from the PHP element's PHPDoc.

Signature

The PHP element's return type.

Classes / Interfaces

Hovering over a class / interface reference presents the location of the class / interface and a short description.

Functions

Hovering over a function call displays the function's location, its signature (including the return type) and the short description.

Class variables

Hovering over class variables displays the function's location and the short description.

  1. Hover over the class reference ($sugar variable in the last line). The tooltip is displayed and includes the following:

    • Class name

    • Correct File location where the class is declared

    • Short Description of the class

Global variables

Hovering over a global variable that is defined outside the file will display the variable's location.

 

Consider the following PHP code containing a class (coffee) containing functions and a PHPdoc description:

<?php
//---Start Include ---->
/**
*This variable holds the coffee type
*@var string;
*/
$coffee_type = 'cappuchino';
/**
* This class makes coffee preparations
*/
class coffee{   
 public $sugar;
 /**
 * This variable holds sugar portion for teaspoon in grams
@var int
 */
 private $tea_spoon;
 function coffee($sugar){  
    /* Cappucino */  
    $this->sugar = $sugar;
    $this->tea_spoon = 20;
 }    
 /**   
This function returns sugar quantities
 * @return int   
 */   
  function add_sugar(){
     return ($this->sugar > 0)  ? $this->sugar * $this->tea_spoon : 0;
   }
}
$coffee_time=1;
//--- End Include ---->  
$cup = new coffee(2);
$sugar = $cup->add_sugar();
echo $coffee_time;
?>
