Build Paths

The PHP build process scans all files, folders, projects and libraries that are on the project's PHP Build Path so that resources which are referenced in the project can be made available for Code Assist options and Refactoring operations. The required resources must therefore be included in the PHP Build Path.

The project's Build Path properties page allows you to select PHP resources to include/exclude from this process. Rather than automatically scanning all referenced projects and libraries, configuring the Build Path allows you to select which resources will be scanned, thereby significantly speeding up the build process.

 

The Build Path is determined by the resources included in the project's Build Path  properties page.

 

 

To configure the project's Build Path:

  1. In PHP Explorer view, right-click the required project and select Build Path | Configure Build Path.
    The Build Path properties page is displayed.

This page allows you to include/exclude specific files, folders and variables from the Build Path.

 

To add a source folder from your Workspace to the Build Path:

  1. Click Add folder.
    The Source Folder Selection dialog will display.

  2. Select the required folders and click OK.

 

You can choose to include or exclude specific resources or file name patterns from within source folders which you have added.

To include/exclude specific resources from the Build Path:

  1. Expand the node next to the folder from which you would like to include/exclude resources.
    A list of the resources which are included and excluded will be displayed (by default, all resources within the folder are included).

  2. Select the Included or Excluded list and click Edit.
    The Inclusion / Exclusion patterns dialog is displayed.

  1. Click Add next to the Inclusion or Exclusion patterns panes.
    The Add Inclusion/Exclusion Pattern dialog is displayed.

  2. Enter or select the required resources / pattern to Include/Exclude and click OK.

  3. Click Finish.
    All resources in the folder which match an Inclusion pattern but do not match an Exclusion pattern will be added to the Build Path.

Saving your changes

Once you have made all necessary changes to your Build Path, click OK.

Your project will be rebuilt to reflect the changes, according to the settings you configured.
